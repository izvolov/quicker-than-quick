# Быстрее быстрой, или Как обогнать `std::sort` с помощью поразрядной сортировки

https://izvolov.github.io/quicker-than-quick/

Алгоритм поразрядной сортировки незаслуженно попал в разряд "академических", то есть полезных в теории, но неприменимых на практике.

В своём выступлении покажу, что этот алгоритм чрезвычайно эффективен и полезен, и докажу это на примере его внедрения в `std::stable_sort`.
